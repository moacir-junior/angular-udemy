Serviços
- Classes para injetar em componentes e em outros serviços
- Usados para encapsular acesso a APIs de back-end
- Podem ser singleton ou parte de aplicação
- Escopos:
	- Módulo: Disponível para todas as classes declaradas no módulo - providers: []
	- Componentes e seus filhos: Disponível para o componente e seus componentes filhos - providers: []
	- Componente: Somente Componente - viewProviders: []


Pipes
- Transformam conteúdos
- Usamos: propriedade | Pipe
- Para passar parâmetros para um Pipe usamos ":" - propriedade | Pipe: parâmetro
- Exemplos:
	- json
	- uppercase
	- lowercase
	- percent
	- date: formato
	- currency: iso
	- slice: limite
- Podemos combinar pipes com outros
